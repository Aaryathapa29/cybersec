# Cyber Security Lab

## Overview

This repository contains the practical assignments completed for the Cyber Security laboratory course. The programs demonstrate the implementation of classical cryptographic techniques, decryption methods, and basic cryptanalysis using Python.

## Practicals Included

### Practical 2.1 – Caesar Cipher

* Caesar Cipher encryption
* Caesar Cipher decryption
* Sender and receiver communication
* Key generation using roll number modulo 26

### Practical 2.2 – Brute Force Attack

* Brute force attack on the Caesar Cipher
* Tries all possible keys (1–25)
* Displays every possible plaintext
* Identifies the correct key by observing meaningful output

### Practical 2.3 – Frequency Analysis

* Standard Frequency Distribution (SFD)
* Single-letter frequency analysis
* Digram analysis
* Trigram analysis
* Initial substitution table
* Partial decryption using frequency matching

## Technologies Used

* Python 3
* Visual Studio Code
* Git
* GitHub


## Learning Outcomes

* Understand classical encryption and decryption techniques.
* Implement the Caesar Cipher in Python.
* Perform brute-force cryptanalysis.
* Analyze ciphertext using frequency analysis.
* Understand the importance of key security.
* Gain hands-on experience with basic cryptographic concepts.
